O repositório é uma implementação de um trabalho de Introdução à Teoria dos Grafos.

No arquivo `ArvoreDeDecisaoNeural.py` é implementado uma classe com o algoritmo da árvore de decisão, que faz a inferência equivalente de uma rede neural.

O notebook jupyter `turning-nn-into-dt.ipynb` possui três testes para verificar se os resultados de inferência de redes neurais treinadas por meio do tensorflow são realmente equivalentes as das árvores de decisão.
